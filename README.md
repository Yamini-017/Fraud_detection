# AI-Powered Credit Card Fraud Detection System

An intelligent fraud detection system that combines multiple Machine Learning and Deep Learning models with a FastAPI backend, Blockchain audit logging, and secure authentication to detect fraudulent credit card transactions in real time.

---

## Features

- Data preprocessing pipeline
- Autoencoder-based anomaly detection
- Isolation Forest anomaly detection
- LSTM deep learning model
- XGBoost classifier
- Ensemble prediction model
- FastAPI backend
- SQLite database integration
- Blockchain audit logging
- JWT Authentication
- Web frontend
- Real-time fraud simulation
